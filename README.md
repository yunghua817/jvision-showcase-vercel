# Jvision Demo 展示館

集中展示 Jvision 各產業與工作情境的互動 Demo，提供關鍵字搜尋、分類篩選與 RWD 瀏覽，訪客可從卡片直接進入正式系統操作。

## 線上展示

[https://jvision-demo-showcase.vercel.app/](https://jvision-demo-showcase.vercel.app/)

## 展示館功能

- 收錄 64 套可操作的 Jvision Demo
- 依產業與情境分為 8 個類別
- 支援系統名稱、功能與產業關鍵字搜尋
- 桌機、平板與手機皆可正常瀏覽
- 每張卡片可直接前往對應的正式 Demo

## 本機執行

```bash
npm install
npm run dev
```

正式建置：

```bash
npm run build
npm run start
```
